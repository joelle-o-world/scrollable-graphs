Class MarkerGraph

The MarkerGraph class represents a series of unlabelled time-points.

e.g./

// Create a marker graph
let markers = new MarkerGraph([570, 1789, 1848, 1865, 1991, 1994]);

// Render the graph
let view = new TrackView;
view.addGraph(markers);
view.draw()
